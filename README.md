## 安裝

Node.js 版本建議為：`16.15.0` 以上...

### 取得專案

```bash
git clone git@github.com:kaikai0517/modern-office.git
```

### 移動到專案內

```bash
cd modern-office
```

### 安裝套件

```bash
npm install
```

### 運行專案

```bash
npm run dev
```
